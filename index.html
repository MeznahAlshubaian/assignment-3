16-bit Pipelined CPU

Designed and implemented a 16-bit pipelined processor with custom RISC instructions for COE301.

This project involved building a single-cycle 16-bit CPU with a 5-stage pipeline, including instruction fetch, decode, execute, memory, and write-back stages. The CPU supports arithmetic, logical, branching, and memory instructions. I tested hazard detection, forwarding, and pipelined control logic. The project demonstrates knowledge of CPU architecture, instruction sets, and performance optimization techniques.
